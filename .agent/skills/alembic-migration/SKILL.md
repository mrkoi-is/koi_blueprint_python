---
name: alembic-migration
description: Generate, review, and apply Alembic migrations in a Koi-standard Python service. Use when models change, when you need a new revision, or when you must inspect autogenerated upgrade and downgrade logic before applying it locally.
---

Workflow:
1. Confirm `migrations/env.py` points at the correct metadata.
2. Run `uv run alembic revision --autogenerate -m "<message>"`.
3. Review the generated revision manually.
4. Run `uv run alembic upgrade head` locally.
5. Keep production migration execution in deployment pipelines.

Load `references/migration-checklist.md` for the review checklist.
