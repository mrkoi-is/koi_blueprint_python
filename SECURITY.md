# Security Policy

## Supported Versions / 受支持版本

| Version | Supported          |
|---------|--------------------|
| 0.2.x   | ✅ Yes             |
| < 0.2   | ❌ No              |

## Reporting a Vulnerability / 报告安全漏洞

**Please do NOT report security vulnerabilities through public GitHub issues.**

If you discover a security vulnerability in this project, please report it responsibly:

1. **Email**: Send details to **security@mrkoi.is** (or the maintainer's contact listed in the repo).
2. **GitHub Security Advisory**: Use [GitHub's private vulnerability reporting](https://github.com/mrkoi-is/koi_blueprint_python/security/advisories/new).

**请勿通过公开 Issue 报告安全漏洞**，请使用上述私密渠道。

### What to include / 报告内容

- Description of the vulnerability / 漏洞描述
- Steps to reproduce / 复现步骤
- Impact assessment / 影响评估
- Suggested fix (if any) / 建议修复方案（如有）

### Response timeline / 响应时间

- **Acknowledgment**: Within 48 hours / 48 小时内确认收到
- **Assessment**: Within 7 days / 7 天内评估
- **Fix release**: Best effort within 30 days / 尽力在 30 天内发布修复

Thank you for helping keep this project secure! 感谢你帮助保持本项目安全！
